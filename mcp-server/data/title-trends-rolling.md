## 2026-07-21

### 趋势 1: 自我代入式标题成抖音爆款主力
- 案例标题:"我收到高考录取通知书了"(抖音#4,1096万热度)
- 案例标题:"我在豆包养宠物"(抖音#5)
- 套路:新句式(短视频叙事范式入侵)
- 适用平台:抖音 ★★★ / 小红书 ★★★ / 公众号 ★★ / 微博 ★
- 备注:从短视频口播演化的"第一人称当下时"标题,模仿成本低,刷到停留率高

### 趋势 2: 一句话造句反转稳态成为微博热搜主导句式
- 案例标题:"坚持不结婚的人,最适合结婚"
- 案例标题:"终于理解奶奶辈的衣服了"
- 案例标题:"见识到三伏天的厉害了"
- 套路:老套路变体(冲突对比的简化版)
- 适用平台:微博 ★★★ / 抖音 ★★★ / 小红书 ★★★
- 备注:完整可独立成句,自带反转,50% 微博热搜标题都是这种结构

### 趋势 3: 年龄/性别/身份反转成为新闻类标题新爆款引擎
- 案例标题:"26岁女子恋爱后体检发现自己是男性"(微博+头条+知乎)
- 案例标题:"92岁独居老人离世留下500万遗产"
- 案例标题:"33岁外卖员脑梗倒地多日才被发现"
- 套路:身份标签式 + 颠覆认知的合体
- 适用平台:微博 ★★★ / 头条 ★★★ / 知乎 ★★★ / 抖音 ★★
- 备注:慎用于小红书,易被认为贩卖焦虑

### 趋势 4: AI 焦虑/AI 互动成为新平台话题链
- 案例标题:"AI 都扛不住反复焦虑的人"(微博#20)
- 案例标题:"我在豆包养宠物"(抖音#5)
- 案例标题:"AI 的猴子为什么能让菲律宾破防"(头条)
- 案例标题:"OpenAI 负责人称 Kimi 开源是减速主义"(知乎)
- 套路:AI 互动 + 反差/打脸 的新组合
- 适用平台:全平台,但生命周期可能半年内被新词替代
- 备注:深度话题(知乎"AI 最后一公里")会下沉为情绪钩子(微博"AI 焦虑")是当前跨平台迁移稳定路径

### 趋势 5: 名词当形容词 / 高级感量化在多平台扩张
- 案例标题:"野狗骨头 震撼美味"(微博 21万 + 抖音 + 头条)
- 案例标题:"白色海莉美甲的高级感超标了"(抖音#11)
- 案例标题:"雪花多舍不得冬天"(抖音)
- 套路:非常规形容词 / 高频形容词 + 超出预期的副词
- 适用平台:小红书 ★★★ / 抖音 ★★★ / 微博泛话题 ★★
- 备注:短剧/直播切片带火的范式,"封神/震撼/超标/高级"等词正在取代"OMG/绝了"

### 趋势 6: B站空格/全角符号节奏切割成综艺鬼畜新范式
- 案例标题:"《 双 枪 牛 仔 》"
- 案例标题:"⚡对 对 子 战 神 8⚡"
- 案例标题:"全 网 最 幸 运 的 动 画 教 程 3.5"
- 套路:B站独有视觉节奏
- 适用平台:B站 ★★★ / 抖音 ★★ / 小红书 ★★
- 备注:公众号慎用,显得 low;但视觉切割在短视频封面中非常醒目

### 趋势 7: 头条官方 label 标签体系正被自媒体跟进
- 案例标签:[hot] / [new] / [refuteRumors] / [interpretation] / [recentProgress]
- 案例:"[hot] 中方紧急召见菲驻华大使"
- 案例:"[refuteRumors] 广西辟谣女流浪汉怀孕"
- 套路:权威认证感 + 一句话标题
- 适用平台:头条 ★★★ / 微博 ★★ / 知乎(知乎用"如何看待"前缀)★★ / 公众号 ★★
- 备注:主持人/评论员/媒体 三个官方身份前缀也是这一波范式

### 趋势 8: 跨平台同热事件每个都能做出多版本标题
- 谢贤去世 → "从杨过到赌神 谢贤的传奇一生"(头条[hot])/"谢贤生前最讨厌哪种告别"(头条[hot])/"谢贤进 ICU 强撑到谢霆锋回港"(头条)
- 野狗骨头 → 多平台引用同一句"震撼美味",各自补完不同身份前缀
- 怡颗莓致癌物 → 微博#1 实测数据 → 头条[recentProgress]跟进品牌回应

### 趋势 9: 老套路新变体
- 震惊体已死 / 不转不是中国人死 / 98% 不知道有反噬
- 数字+身份已演化为数字+性别/年龄反转
- "辟谣!"已演化为平台自带[refuteRumors] label
- "N大X"已演化为"我X了""我把X做了"自我代入

## 当前规则(2026-07-21 提取)

**5 个上调权重的套路**:
1. 自我代入(尤其第一/三人称当下时)
2. 一句话造句反转(尤其"坚持 X 的 X,最 Y")
3. 年龄/性别/身份反转(尤其真实案例)
4. AI 互动/AI 焦虑
5. 名词当形容词 / 高级感量化

**3 个新平台特有**:
- 头条官方 label 标签
- B 站空格节奏切割
- 抖音第一人称当下时

**5 个应剔除的过时套路**:
1. 震惊体
2. 不转不是中国人
3. 98% 的人都不知道
4. 极限词滥用(无资质)
5. 极端化悬念结论("可怕""惊人""震惊")
