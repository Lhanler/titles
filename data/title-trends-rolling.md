## 2026-07-21

### 趋势 1: 自我代入式标题成抖音爆款主力
- 案例标题:"我收到高考录取通知书了"(抖音#4,1096万热度)

- 适用平台:抖音 ★★★ / 小红书 ★★★ / 公众号 ★★ / 微博 ★
- 备注:从短视频口播演化的"第一人称当下时"标题,模仿成本低,刷到停留率高

### 趋势 2: 一句话造句反转稳态成为微博热搜主导句式

- 适用平台:微博 ★★★ / 抖音 ★★★ / 小红书 ★★★
- 备注:完整可独立成句,自带反转,50% 微博热搜标题都是这种结构

### 趋势 3: 年龄/性别/身份反转成为新闻类标题新爆款引擎

- 适用平台:微博 ★★★ / 头条 ★★★ / 知乎 ★★★ / 抖音 ★★
- 备注:慎用于小红书,易被认为贩卖焦虑

### 趋势 4: AI 焦虑/AI 互动成为新平台话题链

## 当前规则(2026-07-21 提取)

**5 个上调权重的套路**:
1. 自我代入(尤其第一/三人称当下时)
2. 一句话造句反转(尤其"坚持 X 的 X,最 Y")
3. 年龄/性别/身份反转(尤其真实案例)
4. AI 互动/AI 焦虑
5. 名词当形容词 / 高级感量化

**3 个新平台特有**:
- 头条官方 label 标签
- B 站空格节奏切割
- 抖音第一人称当下时

**5 个应剔除的过时套路**:
1. 震惊体
2. 不转不是中国人
3. 98% 的人都不知道
4. 极限词滥用(无资质)
5. 极端化悬念结论("可怕""惊人""震惊")
