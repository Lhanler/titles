# 平台标题速查表

每个平台都有"生死线"——字数、风控、调性,踩了就废。

---

## 公众号 / 自媒体(订阅号为主)

### 生死线
- **字数**: 15-25 字最佳,超过 30 字在订阅列表里会截断
- **前 8 字决定生死**: 微信订阅号只显示前 18-20 字,核心信息前置
- **封面图**: 一张图就 50% 的点击率,标题+封面强协同

### 风格密码
- **故事感强**: "那年我 25 岁,在地铁里哭成狗"
- **观点明确**: 公众号用户要的是"被表达",不是"被告知"
- **情绪钩子**: 焦虑、共鸣、愤怒、治愈,这四类情绪最易爆
- **疑问/对比**: 自带悬念,易点开

### 高频爆款套路
- "我花了 X 元 / X 年,终于弄懂了 XXX"
- "成年人最体面的拒绝,是 XXX"
- "XXX 背后,藏着 XXX 的真相"
- "这 5 个细节,暴露了 XXX 的人"

### 红线 / 限流规则
- ❌ 含"转发""不转不是中国人"等诱导分享词 — 会被限流
- ❌ 标题与正文严重不符 — 会被投诉+扣信用分
- ❌ 标题带"震惊!"这种 — 时代变了,显得 low
- ❌ 标题党、淫秽色情、虚假医疗 — 直接封号

### 头条 vs 二条
- 头条标题要更狠、更反常,流量基数大
- 二条标题可以更温和,因为点开的人已是有粉

## 抖音 / 短视频(视频号 / 快手 / B站通用)

### 生死线
- **字数**: 12-18 字,**超过 22 字的标题在信息流里会被截**
- **首帧 1 秒定生死**: 标题和封面+前 3 秒共同决定是否完播
- **互动设计**: 标题里带问号、争议词,易引发评论

### 风格密码
- **强冲突**: "室友 vs 室友""前任 vs 现任""专家 vs 大妈"
- **强反差**: 反常识 + 反身份的组合
- **强情绪**: 惊讶、质疑、共鸣、愤怒
- **悬念前置**: 把"答案"藏到视频里

### 高频爆款套路
- "XXX?我试了,结果…"
- "99% 的人都搞错了 XXX"
- "XXX 的真相,看完沉默了"
- "被 XXX 骗了 X 年"
- "XXX 不可怕,可怕的是 XXX"

### 红线 / 限流规则
- ❌ 标题党: 视频内容必须能兑现标题
- ❌ 政治敏感、低俗色情、未成年人负面
- ❌ "最""第一""独家"等极限词,如无资质会限流
- ❌ 故意制造对立(如地域、性别),易被标记

### 平台差异化
- **抖音**: 节奏快、悬念强、信息密度高
- **视频号**: 受众更熟龄,故事感、情感共鸣更有效
- **快手**: 老铁文化、真实感,夸张表演
- **B站**: 知识密度+梗,适合深度+二刷

## 小红书

### 生死线
- **字数**: 20 字以内,**首屏可见前 13 字,核心信息前置**
- **emoji 不是装饰是语义符号**: 🆘🚨✨ 是个小标题
- **tag 和 emoji 是流量入口**: 搜索逻辑权重高

### 风格密码
- **存钱/避雷/急救**: 三类最容易爆
- **真实感 + 调侃**: "姐妹们救命""谁懂啊""破防了"
- **强代入感**: "我愿称之为救命神器"
- **对比感**: "用前 vs 用后""以前的我 vs 现在的我"

### 高频爆款套路
- "救命!这个真的 yyds"
- "姐妹们冲!XX 真的太香了"
- "谁懂啊!XXX 终于让我摆脱了 XXX"
- "用了 X 年 X 元,亲测这 X 个值得入"
- "内行人真心建议:XXX"

### 红线 / 限流规则
- ❌ 站外引流(微信号、淘宝链接、二维码) — 直接限流
- ❌ 营销词"最低价""第一" — 平台审核严格
- ❌ 笔记和图片严重不符
- ❌ 医学建议、医美项目过度承诺

### 搜索逻辑
小红书是搜索引擎,**标题里堆关键词 = 抢搜索流量**。包含:
- 品类词(防晒霜/咖啡机)
- 受众词(学生党/打工人/小个子)
- 效果词(显瘦/油皮救星)
- 场景词(通勤/约会/面试)

## 微博 / 微博热搜(2026 起已成标题风向标)

### 生死线
- **字数**:无硬上限,但热搜词条 4-18 字最常见,超过会被截断成两行影响点击
- **节奏**:热搜词条本身不能带标点(空格、问号都少见),更像"标签"而不是"句子"
- **同质化严重**:同事件常出现多个近似词条,标题党会蹭多个词

### 风格密码
- **造句式 > 惊叹号式**:微博热搜标题 60%+ 是"可以独立成立的一句话"
  - 例:"终于理解奶奶辈的衣服了"(49 万)/ "坚持不结婚的人最适合结婚"(25 万)
- **第一人称 + 完成时**:自我代入体天然适合微博
  - 例:"我将逐张学习章若楠的游客照"
- **AI 焦虑关键词**:2026 起,带 AI/算法/大模型词的标题上升明显

### 高频爆款套路
- "X 是 Y"反差词(野狗骨头 震撼美味)
- "X 岁 X 发现自己是 Y"身份反转
- "短视频梗反哺新闻"口语化造句
- "被 X 改写了/重新定义"行业复盘

### 红线 / 限流规则
- ❌ 含"震惊""不转不是中国人"等诱导分享词
- ❌ 涉政敏感、未成年负面、明星隐私
- ❌ 蹭明星私事(八卦号常见被封原因)
- ❌ 营销号批量搬运无加工

### 数据获取方式
微博热搜是**最稳定的公开热榜**之一,API 直拉即可,无需登录:
```bash
curl -H "Referer: https://weibo.com" "https://weibo.com/ajax/side/hotSearch"
```
详见 `references/live-data-sources.md` 第 1 节。

## 今日头条 / 头条号(自带官方 label,公式化最强)

### 生死线
- **字数**:微头条 18-25 字最常见,头条文章标题 18-22 字
- **信息密度**:头条用户"刷新闻"目的强,标题必须 1 句话讲清"发生了什么 + 为什么看"
- **官方 label 是钩子**:`[hot] [new] [refuteRumors] [interpretation] [recentProgress]` 五个标签本身就是分类器

### 风格密码
- **评论员前缀**:"评论员:" "媒体:" "观察:" "分析:"是头条独有体,提升权威感
- **深 V 反转类**:股市/经济类标题模板"A 股深 V 反转""暴力反弹""绝地大反转"
- **辟谣+主语**:`[refuteRumors] XX 辟谣"XX"` — 直接用 label 就赢一半
- **解读体**:`[interpretation] 为什么 XX 此时 XX` — 适合深度文

### 高频爆款套路
- "[hot] 重大事件!XX" — 蹭官方 hot 标签
- "XX 这轮 XX 结束了吗?" — 9.6 提问体
- "XX 终于 XX" — 长期事件的爆点(如"科技股终于暴力反弹")
- "XX 万 + 用户在用的 XX" — 数据驱动型

### 红线 / 限流规则
- ❌ 标题党/正文不符 — 头条 AI 审核严,扣信用分
- ❌ 涉政、医疗健康、财经未认证
- ❌ 标题带"震惊""速看""转发"

### 数据获取方式
```bash
curl "https://www.toutiao.com/hot-event/hot-board/?origin=toutiao_pc"
```
详见 `references/live-data-sources.md` 第 2 节。

## B 站(空格标题 / 标点延展是 2025-2026 新稳定范式)

### 生死线
- **字数**:无硬上限,但 8-25 字最常见;B 站是"标题 + 封面"协同,标题本身可以"留白"
- **节奏切分**:B 站标题独特之处 — **用空格/标点把标题切成节奏块**,降低视觉密度反而抓眼

### 风格密码
- **空格标题**(2025-2026 稳定范式):
  - 《 双 枪 牛 仔 》
  - ⚡对 对 子 战 神 8⚡
  - 全 网 最 幸 运 的 动 画 教 程 3.5
- **梗文化密度**:UP 主标题自带"懂的人自然懂"的暗号感
- **学习类 + 数字教程**:"十年老师傅无保留教大家""67 岁老人坚持锻炼教程"

### 高频爆款套路
- "十年 X 师傅,无保留教你 X" — 权威感 + 数字
- "X 岁 X 坚持 X 教程" — 反差人物 + 教程
- "⚡X X X X⚡" — emoji 包裹节奏块
- "X 教程 / 完整版 / 上线" — 知识类
- "(X)" 后缀补充 — "(完整版)""(官方录播)"

### 红线 / 限流规则
- ❌ 标题党/封面党/诱导三连
- ❌ 涉政、低俗、未成年负面
- ❌ 极限词("最""第一")除非有数据

### 数据获取方式
```bash
curl "https://api.bilibili.com/x/web-interface/ranking/v2?rid=0&type=all"
# rid 1=动画 3=娱乐 4=游戏 168=生活
```

## 万能自检表

任何标题生成后,过这一遍:

- [ ] **前 8 字**: 用户刷信息流时,看前 8 字能否让他停?
- [ ] **平台字数**: 没超平台上限?
- [ ] **关键词**: 是否有 2-3 个高潜搜索词?
- [ ] **情绪**: 命中"好奇/焦虑/共鸣/打脸/实用"至少其一?
- [ ] **受众**: 谁会点?点开的人是不是我想要的精准用户?
- [ ] **真实性**: 内容能兑现标题吗?
- [ ] **红线条**: 没有违反平台风控的词?

不过的话,直接砍掉重来。

## 2026-07 平台新观察(由 491 条真实标题调研补充)

### 抖音
- **新风向**: 自我代入式成爆款主力,"我 X 了""我在 X 养 Y""已升为 X"占比显著上升
- **必带元素升级**: 口语化 + 第一人称当下时 + 一个感受词(震惊/破防/封神)
- **慎用**: "真相""惊人""可怕"等抽象词,推荐改成具体画面词

### 微博
- **新风向**: 50%+ 的热搜都是"一句话造句反转"结构,"坚持 X 的 X,最 Y" / "终于理解 X 了"
- **必带元素**: 一句完整可独立成句的话 + 内置反转
- **慎用**: 长定语、多重转折、抽象形容

### 头条
- **新风向**: 官方 label 标签体系正被自媒体跟进,[hot] / [new] / [refuteRumors] / [interpretation] / [recentProgress] 前缀
- **主持人/评论员/媒体权威前缀**: "主持人:""评论员:""媒体:" 是新一波标题范式
- **特别注意**: 头条对无资质极限词"最""第一"会限流

### 知乎
- **新风向**: "AI 最后一公里""Kimi vs DeepSeek V4" 类硬核深度话题在持续输出
- **下沉迁移**: 深度话题会下沉为情绪钩子(知乎"AI 最后一公里" → 微博"AI 都扛不住你")
- **慎用**: 短视频口播风格的标题(会被认为不严谨)

### B 站
- **新风向**: 空格/全角符号做节奏切割已成体系,"《 双 枪 牛 仔 》""⚡对 对 子 战 神 8⚡"
- **综艺鬼畜范式**: 全站热门 100 条中至少 15 条用此手法
- **特别注意**: 公众号/专业内容慎用,容易被认为"low";但短视频封面用,视觉非常醒目

### 小红书(参考其他平台迁移规律)
- **预测**: 自我代入 + 名词当形容词 + AI 焦虑 三套路在小红书将爆发
- **当前最强搜索词**: 显瘦/油皮救星/学生党/通勤/打工人 + 关键词堆砌
- **特别注意**: 年龄/性别/身份反转要慎用,易被认为贩卖焦虑

### 公众号(以少数派/华尔街见闻为代表)
- **新风向**: "我 X 了""第一次做 X""X 天/周/月 + 数字" 的自我代入范式
- **必备**: 真实细节 + 时间量化 + 一个反常识结论
- **范例**: "第一次做开源,39 天 800 Star""中东局势与财报季双重压力下,美股三连跌"

## 新发现的"3 个跨平台同热事件模板"

任何能跨平台同热的事件,都是出多版本标题的好机会:

1. **人物去世类**(以谢贤去世为案例)
   - 头条 [hot] 范式:"从 X 到 X,XXX 的一生"(传奇回顾)
   - 头条 [hot] 范式:"XXX 生前最讨厌哪种告别"(角度切入口)
   - 微博造句范式:"听到 XXX 时,我突然 XXX"(情感冲击)
   - 知乎深度范式:"为何 XXX 在这一代人中地位特殊?"

2. **短剧爆款类**(以野狗骨头为案例)
   - 微博造句范式:"野狗骨头 震撼美味"(名词当形容词)
   - 抖音"已升为"范式:"已升为野狗骨头的死忠粉"
   - 头条 [hot] 范式:"野狗骨头 震撼美味"(同微博,跨平台联用)

3. **品牌翻车类**(以怡颗莓致癌物为案例)
   - 微博【事件】:"怡颗莓 致癌物"
   - 头条 [recentProgress]:"怡颗莓回应检出致癌物"(公关应对滞后)
   - 知乎深度:"怡颗莓品牌是否值得继续信任?"

**通用规则**: 当同一事件跨 3 个+ 平台同热,务必出 3+ 个版本的标题,每个平台 1 个。
